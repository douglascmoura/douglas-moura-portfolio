# 🌐 DOCHMO - Portfólio de Douglas Chaves Moura

Bem-vindo ao repositório do *site* **DOCHMO**. 

Este projeto representa o portfólio oficial que consolida **Estatística Aplicada, Ciência de Dados, Machine Learning e Data Storytelling** em uma experiência visual de altíssimo nível (Premium/Dark Mode).

---

## 🎯 Objetivos do Projeto

O *site* atua como um repositório central unificado para demonstração técnica e estética, construído com o foco em:
1. **Design de Interface Avançado:** Apresentar análises e scripts em um ambiente denso, com tons sóbrios (Azul Noturno e Dourado Metálico), projetando autoridade técnica e prestígio.
2. **Data Storytelling:** Engajar o leitor através de descrições bem construídas e micro-animações, reduzindo a fricção e simplificando o complexo.
3. **Consolidação de Estudos:** Unificar 5 Estudos de Caso de altíssimo nível, indo desde Machine Learning até Regressão Beta para Dados Longitudinais.

---

## 🛠️ Tecnologias e Destaques Metodológicos

A arquitetura do *site* prioriza a leveza de carregamento e um alto rigor estético, abrindo mão de frameworks invasivos e optando por desenvolvimento puro onde o design importa:

* **HTML5 Semântico e Vanilla CSS:** Estruturação através de Flexbox, CSS Grid e Custom Properties para estabilidade visual e escalabilidade do index.html
* **JavaScript & DOM:** Criação de *Custom Cursors*, interceptadores dinâmicos, e loaders temáticos para imersão instantânea do usuário.
* **GSAP (GreenSock) & ScrollTrigger:** Motor utilizado para orquestrar animações baseadas no rolamento da página (Parallax, reveals encadeados e fades responsivos).
* **Gráficos Embarcados (Plotly):** Integração nativa de Mapas Espaciais, Heatmaps de Correlação e Dispersões 3D extraídos diretamente dos relatórios em Python e R para interatividade imediata do avaliador.
